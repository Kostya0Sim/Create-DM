# ![logo](https://github.com/Kostya0Sim/Create-DM/blob/main/pack.png) Create: DM
This resource pack changes the appearance of the [Create](https://www.curseforge.com/minecraft/mc-mods/create) GUI to the darker and less saturated tones like in a [Default Dark Mode](https://www.curseforge.com/minecraft/texture-packs/default-dark-mode) resource pack.
